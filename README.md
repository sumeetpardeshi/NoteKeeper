## Getting Started

1. Clone and install dependencies:

```bash
git clone https://github.com/yourusername/notekeeper.git
cd notekeeper
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173` to view the application.
